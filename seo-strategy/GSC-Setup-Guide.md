# Google Search Console Setup Guide

## Verify Domain
- **Method**: DNS Verification (TXT record)
- **Steps**:
  1. Add property in GSC.
  2. Copy TXT record.
  3. Add to DNS provider.
  4. Verify ownership.

## Submit XML Sitemap
- **URL**: `https://furute.in/sitemap.xml`
- **Action**: Submit sitemap in GSC.

## Inspect Homepage
- **URL**: `https://furute.in/`
- **Action**: Use URL Inspection tool to check indexing status.

## Monitor Coverage
- **Report**: Check "Coverage" report for errors and warnings.
- **Action**: Fix any crawl errors or indexing issues.

## Monitor Rich Results
- **Report**: Check "Enhancements" report for rich result performance.
- **Action**: Optimize structured data for better visibility.

## Track Performance Queries
- **Report**: Analyze "Performance" report for search queries and clicks.
- **Action**: Identify high-performing keywords and optimize content accordingly.
