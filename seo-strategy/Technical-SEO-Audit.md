# Technical SEO Audit Checklist

## Core Web Vitals
- **LCP (Largest Contentful Paint)**: < 2.5s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1

## Mobile-First Indexing
- **Responsive Design**: Ensure content is accessible and readable on all devices.
- **Viewport Meta Tag**: `<meta name="viewport" content="width=device-width, initial-scale=1">`

## Canonical Tags
- **Self-Referencing**: `<link rel="canonical" href="https://furute.in/page-url">`
- **Avoid Duplicate Content**: Ensure unique content on each page.

## Schema Validation
- **Structured Data Testing Tool**: Validate all schemas using Google's tool.
- **Rich Results Test**: Check for rich snippet eligibility.

## Structured Breadcrumbs
- **BreadcrumbList Schema**: Implement breadcrumb schema for clear navigation.
- **Navigation**: Ensure breadcrumbs are visible and functional.

## Sitemap Submission
- **XML Sitemap**: Submit `sitemap.xml` to Google Search Console.
- **Robots.txt**: Reference the sitemap in `robots.txt`.

## Robots.txt
- **Allow Crawling**: Ensure search engines can access important pages.
- **Disallow**: Block irrelevant or sensitive directories.
